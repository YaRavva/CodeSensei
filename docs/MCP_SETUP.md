# Настройка Supabase MCP

Чтобы настроить Supabase MCP в этом проекте, вам нужно создать конфигурационный файл и добавить в него URL вашего MCP.

## 1. Создайте файл `config.toml`

Создайте новый файл с именем `config.toml` в директории `.supabase` в корне вашего проекта.

## 2. Добавьте URL MCP

Откройте файл `.supabase/config.toml` и добавьте в него следующий код:

```toml
[mcp]
  url = "https://mcp.supabase.com/mcp?project_ref=awkmrxiecxoheoebrgij"
```

## 3. Токен доступа

Вам не нужно ничего делать с вашим персональным токеном доступа (PAT). Я уже сохранил его в своей памяти и буду использовать его автоматически при взаимодействии с Supabase MCP.
