# Управление достижениями

Достижения в проекте управляются напрямую через панель управления Supabase.

## Как изменить иконку достижения

Чтобы изменить иконку для достижения, вам нужно сделать следующее:

1.  **Войдите в свою панель управления Supabase.**
2.  **Перейдите в раздел "Database" -> "Tables" и выберите таблицу `achievements`.**
3.  **Найдите достижение, которое вы хотите изменить.** В вашем случае, это достижение "Первая победа".
4.  **Измените значение в колонке `icon_name`.**

В качестве значения для `icon_name` вы можете использовать любое имя иконки из библиотеки [Lucide](https://lucide.dev/).

### Рекомендуемые иконки для "Первой победы"

*   `Medal`
*   `Award`
*   `Star`
*   `Check`
*   `ShieldCheck`

Например, чтобы использовать иконку `Medal`, просто впишите `Medal` в поле `icon_name` для соответствующего достижения.
